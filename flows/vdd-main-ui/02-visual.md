## State Variations Summary

| Screen | State | Condition | Visual Change |
|--------|-------|-----------|---------------|
| Main | Disconnected | Default | Power button shows "Connect" |
| Main | Connected | VPN active | Timer, stats, button shows "Disconnect" |
| Main | No Stat Bar | SHOW_STAT_BAR=false | Stat bar hidden |
| Apps | Enabled | SHOW_APPS_PAGE=true | App list shown |
| Apps | Disabled | SHOW_APPS_PAGE=false | Placeholder, nav icon dimmed |
| Settings | Enabled | SHOW_SETTINGS_PAGE=true | Settings shown |
| Settings | Disabled | SHOW_SETTINGS_PAGE=false | Placeholder, nav icon dimmed |
| Onboarding | Mandatory | No subscription | Skip button hidden/disabled |
| Onboarding | Optional | Subscription configured | Skip button enabled |
